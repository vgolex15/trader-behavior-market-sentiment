# Trader Behavior vs Market Sentiment

## Objective
To analyze the relationship between Bitcoin market sentiment (Fear & Greed Index)
and trader performance using historical Hyperliquid trading data.

## Datasets
- Bitcoin Fear & Greed Index
- Hyperliquid historical trader data  
(Data sourced from links provided in the assignment)

## Analysis
The analysis is performed in a Jupyter Notebook (`analyst.ipynb`) and includes:
- Data cleaning and preprocessing
- Merging sentiment data with trader activity
- Evaluation of PnL, leverage, and trading behavior across Fear and Greed regimes

## Key Insights
- Trader performance varies significantly across market sentiment phases
- Higher leverage usage is more common during Greed periods
- Risk-adjusted returns tend to be more stable during Fear phases

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
